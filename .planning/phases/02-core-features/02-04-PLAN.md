       - setJsonText: (text: string) => void

       - expandNode: (id: string) => void

       - jsonText: '{}'
       - parsedJson: {}
       - nodes: []
       - edges: []
       - selectedPath: null
       - parseError: null
       - isDirty: false
       
       Actions:
       
       setJsonText(text):
         - set({ jsonText: text, isDirty: true })
         - Try parse JSON:

             - Transform to graph: jsonToGraph(parsed)
             - Layout: layoutGraph(nodes, edges)
             - Update: set({ parsedJson: parsed, nodes: layoutedNodes, edges, parseError: null })
           - If error:
             - set({ parseError: error.message })
             - Keep previous nodes/edges (don't clear on error)
       
       selectPath(path):
         - set({ selectedPath: path })
         - Calculate line from path: pathToLine(get().jsonText, path)
       
       updateNodePosition(id, position):
         - Update node position in nodes array
         - set({ nodes: updatedNodes })
       
       expandNode(id), collapseNode(id):
         - Mark node as expanded/collapsed
         - Re-render children

       - getSelectedNode: () => CanvasNode | undefined
       - getErrorCount: () => number
    
    9. Create useAppStore hook and export AppState type.
