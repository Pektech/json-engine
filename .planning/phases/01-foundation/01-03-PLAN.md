# Plan 01-03: Implement File Manager

## Goal
Implement file operations using File System Access API with fallback support, including tests.

## Context
JSON.engine needs to open and save openclaw.json files from the local file system. The File System Access API provides native file operations but requires fallbacks for browsers that don't support it (Firefox, Safari partial). Need to handle permissions, recent files list, and file watching.

## Must Do
- [ ] Create FileHandle type definitions
- [ ] Implement openFile() function:
  - Try File System Access API first
  - Fallback to traditional file input
  - Return file handle, content, and metadata
- [ ] Implement saveFile() function:
  - Save to existing file handle if available
  - Show save dialog for new files
  - Handle permission re-requests
- [ ] Implement recent files tracking:
  - Store in localStorage
  - Track last 10 files
  - Store file name, path, last opened date
- [ ] Implement file watching (where supported):
  - Watch for external changes
  - Show "file changed" notification
  - Offer to reload
- [ ] Write unit tests for all file operations (80%+ coverage)
- [ ] Add error handling for all failure cases
- [ ] Create file manager hook (useFileManager)

## Must Not Do

- Do NOT implement backup logic yet (Phase 3)

## Technical Details

### File System Access API
```typescript
interface FileSystemFileHandle {
  getFile(): Promise<File>;
  createWritable(): Promise<FileSystemWritableFileStream>;
}

// Opening a file
const [fileHandle] = await window.showOpenFilePicker({
  types: [{
    description: 'JSON files',
    accept: { 'application/json': ['.json'] }
  }]
});

// Saving a file
const writable = await fileHandle.createWritable();
await writable.write(content);
await writable.close();
```

### Fallback for Unsupported Browsers
```typescript
// Traditional file input for open
const input = document.createElement('input');
input.type = 'file';
input.accept = '.json';
input.onchange = (e) => { /* handle file */ };

// Download link for save
const blob = new Blob([content], { type: 'application/json' });
const url = URL.createObjectURL(blob);
const a = document.createElement('a');
a.href = url;

```

### Recent Files Format
```typescript
interface RecentFile {
  id: string;           // file handle ID or path
  name: string;         // filename
  path?: string;        // full path if available
  lastOpened: string;   // ISO date
  handle?: FileSystemFileHandle; // stored for re-open (if supported)
}
```

### Error Cases to Handle
- User denies permission
- File not found
- Disk full
- File locked by another process
- Invalid JSON content
- Browser doesn't support File System Access API
